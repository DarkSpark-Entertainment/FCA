# Assets
Considerably the `main` directory of any Unity Project, this folder contains the primary files that are interacted with to develop and run the application, as well as the meta data associated with
all those files.

## [Scripts](/Assets/Scripts/)
This folder contains the major working components for the project, organized as `Core`, `Data`, and `UI`. 

## [Scenes](/Assets/Scenes/)
This folder possesses all scenes in the project, you can find in each scene file the meta data used to put together the Unity UI elements and how they interact with the scripts made for the project.
