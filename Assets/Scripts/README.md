# Scripts
Inside this folder (considerably the `main` directory) contains the major working components for the project, organized as `Core`, `Data`, and `UI`. 

## [UI](/Assets/Scripts/UI/)
The `UI` folder contains scripts that are intended to interact directly with users, usually containing files with the postfix `Controller` to denote the functionality. 

## [Data](/Assets/Scripts/Data/)
`Data` contains scripts that deal with the ingest and egress processes with data, including serialization. 

## [Core](/Assets/Scripts/Core/)
`Core` contains scripts that deal with fundamental background logic to the total application.

### Disclaimer
*Generally, anyone perusing this repository won't find anything of interest (that isnt already in Unity's own API documentation [https://docs.unity3d.com/6000.0/Documentation/ScriptReference/index.html]) outside of the `/Scripts` and `/Scenes` folders.*
